# 💱 Currency Converter (React)

A simple and responsive currency converter built using **React** and a live exchange rate API. It allows users to convert amounts between different currencies in real time.

---

## 🚀 Features

- 🌍 Convert between multiple currencies  
- 🔄 Real-time exchange rates using API  
- 🎯 Simple and clean UI  
- ⚡ Fast and responsive  
- 🔁 Dynamic currency selection  

---

## 🛠️ Tech Stack

- React (Vite)  
- JavaScript (ES6)  
- Tailwind
- Exchange Rate API
